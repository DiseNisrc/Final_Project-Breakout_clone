## 功能說明
a. 七段顯示器顯示累計得分數 \
b. LED燈顯示血條 \
c. 黃色障礙物來阻擋球 \
d. 每打到磚塊得一分 \
e. 除原本的球之外，會偶爾出現特殊球(粉紅球)無視障礙物直接打到磚塊 \
f. 可手動調整球速 \
g. 勝利時會顯示WIN，輸了會顯示LOSE \
h. 打到磚塊會發出聲音
